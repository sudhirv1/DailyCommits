# GigRepository

A repository for tracking and managing gig/freelance work.

## Overview

This repository organizes gig-based projects, tasks, and deliverables to keep your freelance work structured and easy to manage.

## Structure

```
GigRepository/
├── gigs/           # Individual gig project folders
├── templates/      # Reusable templates for proposals and contracts
└── tracking/       # Progress and status tracking for active gigs
```

## Getting Started

1. Copy a template from `templates/` to `gigs/<project-name>/`
2. Fill in the project details
3. Track progress in `tracking/`

## Gig Lifecycle

1. **Proposal** – Use `templates/proposal-template.md`
2. **Active** – Move to `gigs/<name>/` and track in `tracking/status.md`
3. **Completed** – Archive under `gigs/<name>/archive/`
